Flashcard site for learning Js and JQuery

  • Javascript

    Javascript is a programming language that adds interactive features on a webpage for the user.

  • Operators

    Operators are used to perform operations on data types eg. add numbers.

  • Strings

    The String object is used to represent and manipulate a sequence of characters. Strings are used for holding data that could be presented as text form.

  • Methods

    A JavaScript method is a property containing a function definition.

  • Functions

    A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).
